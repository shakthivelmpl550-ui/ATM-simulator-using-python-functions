# 🏧 ATM Simulation System (Python)

A simple Python-based ATM simulation program that allows users to perform basic banking operations like checking balance, depositing money, and withdrawing cash using a PIN-based login system.

## 📌 Features

* 🔐 Secure PIN login system
* 💰 Check account balance
* ➕ Deposit money
* ➖ Withdraw money
* ❌ Handles insufficient balance
* 🔁 Menu-driven program (loop-based interaction)

## 🛠️ Technologies Used

* Python 🐍
* Core concepts:

  * Functions
  * Dictionaries
  * Loops
  * Conditional statements

## 📂 How It Works

1. User enters a **4-digit PIN**
2. If PIN is valid, login is successful
3. User gets ATM menu options:

   * Check Balance
   * Deposit Money
   * Withdraw Money
   * Exit
4. Based on user choice, operations are performed
5. The program continues until the user exits

## ▶️ How to Run

1. Install Python (if not installed)
2. Clone this repository:

   ```bash
   git clone https://github.com/your-username/atm-simulation.git
   ```
3. Navigate to the folder:

   ```bash
   cd atm-simulation
   ```
4. Run the program:

   ```bash
   python atm.py
   ```
